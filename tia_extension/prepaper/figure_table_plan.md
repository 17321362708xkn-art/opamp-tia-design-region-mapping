# Figure And Table Plan

This plan selects a compact figure and table set for a future paper draft. Items marked "main paper" should carry the central story. Items marked "appendix candidate" are useful but may be redundant in a short manuscript.

| ID | Placement | Source file | Purpose | Supported claim | Caption draft | Caution / limitation note |
|---|---|---|---|---|---|---|
