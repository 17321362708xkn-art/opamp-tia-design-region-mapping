## Source Notes

- `ft_proxy_MHz` is a screening parameter for the simplified finite-GBW MATLAB model. For decompensated parts it is the datasheet GBWP and must not be interpreted as unity-gain-stable bandwidth.
- Empty or `NaN` values mean the reviewed datasheet/product summary did not provide a scalar value suitable for this table; they should be confirmed before quantitative ranking.
- Current-noise and input-capacitance values are copied as datasheet screening data and have not been fitted to the MATLAB behavioural model.
- SPICE model availability is recorded only as a planning note. No new vendor macromodel data was imported or compared in Round 7.
