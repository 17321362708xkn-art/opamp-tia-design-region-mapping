# TIA Extension Workspace

This folder is reserved for the planned photodiode TIA behavioural modelling extension.

No TIA model code is added in Round 1.

Future work in this folder should keep the existing active LPF workflow separate from TIA-specific assumptions, scripts, data, and validation outputs.

For a Q3-oriented prototype, this extension must include a real SPICE macromodel comparison module using exported data from selected vendor op-amp macromodels.
